dropthe-go

Go client library for the DropThe open data API. Access 1.83M entities across movies, series, crypto, companies, and people.

About DropThe

This library provides idiomatic Go bindings for the DropThe open data API. DropThe is a data utility media network that aggregates and structures information across entertainment, finance, and technology.

The DropThe platform maintains a knowledge graph of 1.83 million entities with 2.9 million typed relationships, covering movies, TV series, cryptocurrencies, public companies, and notable people. This Go library provides full access to search, entity retrieval, and graph traversal.

# Installation

Go 1.21+ 
# Add to your project
go get github.com/arnaudleroy-studio/dropthe-go

# Import in your code
import "github.com/arnaudleroy-studio/dropthe-go"

# Quick Start

package main

import (
    "fmt"
    "log"

    dropthe "github.com/arnaudleroy-studio/dropthe-go"
)

func main() {
    client := dropthe.NewClient("your-api-key")

    // Search across 1.83M entities
    results, err := client.Search("Christopher Nolan")
    if err != nil {
        log.Fatal(err)
    }

    for _, entity := range results {
        fmt.Printf("%s (%s)\n", entity.Name, entity.Type)
    }

    // Get movie details with linked entities
    movie, _ := client.GetEntity("interstellar-2014")
    fmt.Println(movie.Name, movie.Data["release_year"])
}

# API Reference

GET client.Search(query string) ([]Entity, error)

Full-text search across movies, series, people, companies, and crypto entities.

GET client.GetEntity(slug string) (*Entity, error)

Retrieve a single entity by slug with all metadata and knowledge graph links.

GET client.ListByType(entityType string, opts ...Option) ([]Entity, error)

List entities filtered by type: movies, series, people, companies, cryptocurrencies.

GET client.GetLinks(entityID string) ([]EntityLink, error)

Fetch knowledge graph relationships for an entity (2.9M+ links available).
